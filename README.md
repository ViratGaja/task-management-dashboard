# Task Management Dashboard

A feature-rich task management application built for Kassavirtanen technical interview.

## 🚀 Live Demo
https://task-management-reactjs.netlify.app/

## 🛠️ Tech Stack
- **React** - UI components
- **Redux + Redux Saga** - State management & async operations
- **React Hook Form** - Dynamic form handling
- **Reselect** - Memoized selectors
- **Vite** - Build tool

## ✨ Features
- Create/Edit/Delete tasks with dynamic form fields
- Task types: Bug, Feature, Enhancement, Research
- Redux Saga with optimistic updates & rollback
- Debounced search & multi-filter support
- Auto-save form draft to localStorage
- Normalized Redux state structure

## 📦 Setup
```bash
cd Archive
npm install
npm run dev
```
